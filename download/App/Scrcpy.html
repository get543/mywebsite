ScrCpy (Screen Copy)

Sumber: Genymobile

Aplikasi ini menyediakan tampilan dan kontrol perangkat Android yang terhubung pada USB (atau melalui TCP / IP). Atau juga bisa disebut Screen Mirroing dan hampir tidak ada delay. Itu tidak membutuhkan akses root. Ia bekerja pada GNU / Linux, Windows dan macOS. Tapi link yang disediakan hanya untuk pengguna Windows, jika ingin melihat untuk OS lain bisa cek sumber kodenya ada di Github mereka.


Cara Downloadnya di MyWebsite


  1. Semua aplikasinya bisa kalian liat di drive
  2. Tiggal di klik saja dan dan klik download
  3. Buka Folder Download kalian di File Explorer
  4. Terkadang filenya berbentuk .zip atau .rar, kalian membutukan aplikasi tambahan seperti WinRAR atau 7zip
  5. Jika sudah punya kalian tinggal klik kanan pada filenya lalu pilih 'ekstrak disini'
  6. Double Click pada folder yang abis di ekstrak
  7. Jalankan ***.exe*** file
  8. Jika tidak bisa kalian bisa klik kanan lalu pilih 'Run as Admiinistrator'
  9. Dah kelar..
